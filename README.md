🌐 EventSphere

A modern web-based platform for discovering, managing, and organizing events seamlessly. EventSphere helps users explore events, register easily, and enables organizers to create and manage events efficiently.

🚀 Features
🗓️ Create and manage events
🔍 Browse and search events
👤 User authentication system
📊 Admin dashboard for event approvals
🔔 Real-time notifications (for approvals & updates)
📱 Responsive UI design
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript / React (update if needed)
Backend: Spring Boot
Database: MySQL
Version Control: Git & GitHub
📂 Project Structure
EventSphere/
│── backend/        # Spring Boot backend
│── frontend/       # UI files / React app
│── database/       # SQL scripts
│── README.md
⚙️ Installation & Setup
1️⃣ Clone the repository
git clone https://github.com/Liejox/Eventsphere.git
cd Eventsphere
2️⃣ Backend Setup (Spring Boot)
cd backend
mvn spring-boot:run
3️⃣ Frontend Setup
cd frontend
npm install
npm start
🧑‍💻 Usage
Users can:
View available events
Register for events
Admin can:
Approve or reject submitted events
Manage event listings
